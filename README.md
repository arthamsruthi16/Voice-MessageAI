# 🎙️ Voice-MessageAI Converter

An AI-powered Voice-to-Text Converter that accurately transcribes speech into text with support for multiple languages and customizable output formats. The application provides both real-time voice recording and audio file upload, making speech transcription fast, flexible, and user-friendly.

---

## ✨ Features

### 🎤 Real-Time Voice Recording
- Record your voice directly from the browser.
- Instantly convert spoken words into text.
- Supports continuous speech recognition.

### 📁 Audio File Upload
- Upload audio files for transcription.
- Converts speech from uploaded recordings into text.
- Supports multiple audio formats.

### 🌍 Multi-Language Translation
- Convert transcribed text into your preferred language.
- Choose from multiple supported languages.
- Ideal for multilingual communication and content creation.

### 💬 AI Prompt Box
The application includes an AI-powered prompt box where users can specify how they want the final output.

Examples:
- Convert into paragraphs
- Generate an essay
- Create a conversation
- Summarize the transcript
- Rewrite in a professional tone
- Create meeting notes
- Generate interview transcripts
- Format as bullet points

The AI understands user instructions and formats the output accordingly.

## 📌 How It Works

1. Open the application.
2. Choose one of the following:
   - Record your voice live.
   - Upload an audio file.
3. The application converts speech into text.
4. Select your preferred output language.
5. Enter additional instructions in the AI prompt box.
6. Receive the formatted output instantly.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/Voice-MessageAI.git
```

Navigate into the project

```bash
cd Voice-MessageAI
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```
